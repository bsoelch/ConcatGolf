# ConcatGolf
!!! This is a work in prograss, there can be breaking changes at any time !!!

A concatenative golfing language

## Usage

Build:

```
rustc main.rs -o main
```

Run:

1. create a file `in.txt` containing your source code
2. execute `./main`
